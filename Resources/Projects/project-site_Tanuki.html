    • Cooking System

    I designed the cooking system to be more flexible than a typical crafting system. Dishes do not necessarily require specific ingredients but generic criteria for crafting.

    Recipes contain a list of criteria that the ingredients must meet to be considered for the dish. This allows for a very dynamic and scalable crafting system where the design can accomodate an arbitrary number of rules for what must be input to create a dish.

    A criteria can be defined as specifically requiring a grilled carrot, or more generally, a grilled vegetable, or a carrot of any particular quality. This allows for the design to be very flexible and accomodate a wide variety of crafting possibilities.

    Quest System

    The quest system contains the capability to generate quests dynamically as well as bespoke quests.

    Simulation-Driven World

    The game is meant to feel alive and dynamic. There is a day/night cycle, weather, and seasons. Any actor can contain a Modifier component which will define behavior to update their simulation state.

    An example of this would be the Freshness component.

    The Freshness component will update based on a variety of Influences. Influences are defined and can be defined by things such as temperature, time, and other factors.

    The screenshot here shows the active simulation components and accompanyting influences for an apple on a tree.

    Utility Scoring System

    The utility scoring system is a generic system for implementing weighted-random based decisions in the game. It can be ported across projects easily as it doesn't rely on any specific game mechanics. It is heavily used in the AI system to determine how the NPCs behave.

    NPCs

    The game contains a diverse cast of NPCs that visit the player's restaurant. NPCs contain unique behaviors and personalities that affect how they interact with the player and the world.

    They offer quests, buy items, and provide the player with feedback on their restaurant.

    Building Mechanics

    The game features a building system where the player can construct their restaurant.

    Many structures require the player to fetch resources to build them.

    The player can also upgrade their restaurant to increase the number of customers they can serve.
